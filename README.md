# Pharmacy Management System

## 1. Giới thiệu
Hệ thống quản lý nhà thuốc được xây dựng bằng **Java Spring Boot** và **Hibernate**, hỗ trợ quản lý thuốc, hóa đơn và khách hàng.

## 2. Công nghệ sử dụng
* Backend: Java 17, Spring Boot, Spring Data JPA.
* Database: MySQL / SQL Server[cite: 1].
* Testing: Postman[cite: 1].
